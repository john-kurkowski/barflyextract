=============
barflyextract
=============


Scrape The Educated Barfly's cocktail recipes from its YouTube channel.

Usage
====

```sh
# Your YouTube v3 API key here (perhaps from a dotenv)
export API_KEY=myapikeyhere

make
open build/recipes.html
```
